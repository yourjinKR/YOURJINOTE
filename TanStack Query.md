# TanStack Query

비동기 요청 혹은 **서버 상태**를 관리한다.

> 서버 상태란 데이터베이스에 저장되고 관리되는 데이터를 말한다.

## 지원 기능

- 캐싱
- 중복 요청을 단일 요청으로 통합
- 백그라운드 오래된 데이터를 업데이트
- 데이터가 오래되었는지를 파악
- 데이터 업데이트를 가능한 빠르게 반영
- 페이지네이션 및 데이터 지연 로드와 같은 성능을 최적화
- 서버 상태의 메모리 및 가비지 수집 관리
- 구조 공유를 사용하여 쿼리 결과를 메모화
