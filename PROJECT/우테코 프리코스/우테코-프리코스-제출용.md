# 느낀점
## 꼬리에 꼬리를 무는 공부
특정 개념을 이해하기 위해서 다른 무언가를 계속 공부하는 과정을 거쳤습니다.
또한 직접 연관되어 있지 않더라도 해당 개념에 설명된 예시에서 모르는게 있다면
헤딩 개념 또한 알아봐야 하는 순간들을 경험했습니다.

꼬리에 꼬리는 무는 궁금증으로 인해 순간 *어, 나 뭐하고 있었지?* 라는 생각이 떠오를 때가 있었습니다.
물론 빠른 시간 내에 기본 문법만 습득했기에 기초지식 부족으로 인한 일이라고 생각합니다.
