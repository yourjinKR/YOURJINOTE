2. 나이
3. 성별

~~0. 언어선택 (저희는 영어만)~~

## 퀴즈 방식
- ~~음성 퀴즈~~
- 레벨별로 퀴즈 관리
### 단어 퀴즈 
1. 단어 뜻 맞추기 (영단어를 보고 한글) : 객관식
2. 일치하는 단어 맞추기 (한글을 보고 영단어) : 객관식
### 문장 퀴즈
1. 문장에 빈칸 채우기 : 객관식
2. 단어 조합하여 문장 만들기 : 선택지 주고 순서에 맞게 차례대로 선택
