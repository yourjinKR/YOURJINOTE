# 링크

## 파일명 작성시 주의사항

| **문자**      | **위험한 이유**                                    | **대체 권장**                    |
| ----------- | --------------------------------------------- | ---------------------------- |
| **`[` `](가장%20위험)**%20마크다운%20링크%20문법%20`[%20)`와 충돌하여 링크가 끊김. | `( )` 소괄호 사용                 |
| **`#`**     | URL에서 '헤딩(섹션)'을 가리키는 앵커 기호로 인식됨.              | 사용 금지 (삭제)                   |
| **`&`**     | URL 파라미터 구분자로 오인될 수 있음.                       | `and` 또는 `+`                 |
| **`?`**     | URL 쿼리 시작으로 인식됨.                              | 사용 금지                        |
| **`%`**     | `%20` 처럼 인코딩 문자로 쓰이기 때문에 파일명에 있으면 꼬임.         | 사용 금지                        |
| **`:`**     | 윈도우(Windows) 파일 시스템에서 파일명으로 사용 불가.            | `-` 하이픈                      |
| **`         | `**                                           | 옵시디언 및 마크다운 표(Table) 문법과 충돌. |
