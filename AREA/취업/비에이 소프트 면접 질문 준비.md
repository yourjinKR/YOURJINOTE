외부 API 사용으로는 gpt API, PortOne API를 사용했습니다. 통신은 자바 표준 라이브러리 `java.net.HttpURLConnection`를 사용 **HttpClient**를 사용하지 않았던 이유는 비동기 작업이 필요 없었기 때문입니다.

****개발하면서 어려웠던 점은 API 요청과 응답 과정에서 생기는 결과를 확인하기 어렵다는 점이었습니다. gpt에서도 모니터링 기능을 제공하지만 응답에 대한 내용은 저장하지 않기에 API 요청에 대한 결과 또한 DB에 직접 로깅하는 작업을 진행했습니다. AI 서비스에 대한 자체 버전 관리와 요청에 대한 데이터를 어드민 페이지에 시각화하여 관리할 수 있도록 만들었습니다.

****PortOne API는 여러 pg사를 통합 관리하는 API입니다. 해당 API는 여러 pg사들을 코드의 큰 변화 없이 관리할 수 있기 때문입니다.
