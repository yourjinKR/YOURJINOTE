## 2. Spring Framework / Spring Boot (핵심)

### 2-1. Spring Core
- [ ] Spring이 왜 등장했는지 설명할 수 있다
- [ ] IoC와 DI 개념을 설명할 수 있다
- [ ] Bean의 개념과 생명주기를 이해하고 있다
- [ ] Singleton Scope의 의미를 설명할 수 있다
- [ ] @Component, @Service, @Repository 차이를 알고 있다

### 2-2. Spring MVC
- [ ] 요청 → Controller → Service → Repository → DB 흐름을 설명할 수 있다
- [ ] @Controller와 @RestController의 차이를 설명할 수 있다
- [ ] @RequestMapping, @GetMapping, @PostMapping 등을 사용할 수 있다
- [ ] DTO를 왜 사용하는지 설명할 수 있다
- [ ] @RequestParam과 @PathVariable 차이를 안다

### 2-3. 트랜잭션
- [ ] 트랜잭션의 개념을 설명할 수 있다
- [ ] ACID를 설명할 수 있다
- [ ] @Transactional의 역할을 이해하고 있다
- [ ] 트랜잭션이 Service 계층에 위치해야 하는 이유를 설명할 수 있다

### 2-4. 예외 처리
- [ ] @ExceptionHandler 또는 @ControllerAdvice로 전역 예외 처리를 해본 적이 있다

> 개인프로젝트에서 직접 실습하며 익히기
