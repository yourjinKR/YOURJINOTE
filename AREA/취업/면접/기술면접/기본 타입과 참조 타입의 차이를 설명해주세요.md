# 기본 타입과 참조 타입의 차이를 설명해주세요

Java 언어에 의해 미리 정의되어 있으며, `byte`, `short`, `int`, `long`, `char`, `float`, `double`, `boolean`의 8가지가 존재합니다.  
기본 타입의 변수는 선언된 그 정확한 타입의 **원시 값(Primitive Value) 자체를 메모리에 직접 보유**합니다.

참조 타입은 클래스, 인터페이스, 배열, 타입 변수 타입이 이에 해당됩니다.  
참조 타입의 변수는 데이터 자체가 아니라, 힙(Heap) 메모리에 동적으로 생성된 객체(클래스 인스턴스나 배열)를 가리키는 **포인터(참조값)를 저장**합니다.
