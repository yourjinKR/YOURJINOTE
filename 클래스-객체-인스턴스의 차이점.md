## 클래스, 객체, 인스턴스의 차이는

**클래스**는 **객체를 만들기 위한 설계도**이며 만들어 낼 객체의 속성과 메서드의 집합을 담아놓은 것이다.

**객체**는 해당 **클래스를 기반으로 생성된 대상**입니다.  
클래스로 선언된 변수를 객체라고 합니다.

**인스턴스**는 **객체가 메모리에 실제로 할당된 상태**를 의미하며, 런타임에 구동되는 객체를 말합니다.  
객체와 같은 의미로도 사용됩니다.  

객체와 인스턴스는 관점의 차이로 구분됩니다.

## 부가 설명

```java
class Person {
	int age;
	String name;
	// consturctor, getter, setter...
}
```

`Person` 이라는 객체가 있다고 할때 해당 클래스 타입의 변수가 선언되는 것을 객체라고 말한다.  
인스턴스는 실질적으로 생성자와 같이 실질적으로 메모리에 할당될때 인스턴스라고 한다.  

```java
Person yourjin; // 객체
yourjin = new Person(27, "유어진"); // 인스턴스
```
