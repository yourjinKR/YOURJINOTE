## vscode에서 인식 못하는 현상 

![Pasted image 20260310020424](GALLERY/Pasted%20image%2020260310020424.png)

만약 다수의 레포지토리가 있는 상위 폴더 위치에서 vscode를 열 경우 정상적으로 Github Actions가 작동되는 반면에 `.github` 폴더의 경로가 잘못됐다고 인식하여 경고 메세지가 발생한다.

이와 같은 경우 아래와 같은 해결 방법이 있다.

- vscode 창 새로고침 (Developer: Reload Window)
