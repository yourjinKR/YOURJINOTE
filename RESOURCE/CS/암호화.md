# 암호화

암호화는 권한이 없는 사용자로부터 민감한 정보를 보호하기 위해 읽을 수 있는 텍스트를 읽을 수 없는 암호 텍스트로 변환하는 프로세스이다.

## 대칭 암호화

대칭 암호화는 **단일 키**를 사용하여 데이터를 암호화하고 해독하는 암호화 방법이다.  

- 비대칭 암호화보다 덜 안전
- 처리 능력이 적어 효율적으로 관리 가능

### 대칭키

- 암호화와 복호화에 같은 키를 사용

## 비대칭 암호화

비대칭 암호화는 두 개의 서로 다른 키(공개 키와 개인 키)를 사용하여 데이터를 암호화하고 해독하는 암호화 방법이다.

- 대칭 암호화보다 안전하지만 효율성은 다소 떨어짐

### 비대칭 키

- 공개키 / 개인키 두 개의 키를 사용
- 한쪽 키로 암호화시 다른 쪽 키로만 복호화 가능
