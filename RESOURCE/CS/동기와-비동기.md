# 동기와 비동기의 차이

### 동기 (Synchronous)
- 작업 A가 완료될 때까지 대기(blocking)
- 작업 A가 끝나야만 다음 작업 B를 시작
### 비동기 (Asynchronous)
- 작업 A를 시작하고 완료를 기다리지 않음(non-blocking)
- 작업 A가 진행되는 동안 다른 작업 B, C를 수행 가능

핵심은 **기다림의 방식**이다.

# 동기/비동기 vs Blocking/Non-Blocking
- 동기(Sync): 작업 완료를 기다림
- 비동기(Async): 작업 완료를 기다리지 않고, 나중에 알림받음
- Blocking: 호출된 함수가 제어권을 가져가서, 호출한 함수가 대기
- Non-blocking: 호출된 함수가 즉시 제어권을 반환, 호출한 함수가 다른 일 가능

> 동기/비동기는 작업 완료 시점에 대한 개념이고  
> Blocking/Non-Blocking은 제어권 반환에 대한 개념이다.  
> 실무에서는 Sync-Blocking, Async-Non-Blocking 조합이 가장 많이 쓰인다.
