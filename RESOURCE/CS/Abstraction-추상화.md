# 추상화

불필요한 정보는 숨기고 **중요한 정보만을 표현**함으로써 프로그램을 간단히 만드는 것이다.
자료 추상화를 통해 정의된 자료형을 추상 자료형이라고 한다.
추상 자료형은 자료형의 자료 표현과 자료형의 연산을 캡슐화한 것으로 **접근 제어자**를 통해서 자료형의 정보를 은닉할 수 있다.

## 객체지향 프로그래밍 언어에서

- 클래스 = 추상 자료형
- 객체 = 추상 자료형의 객체
- 메소드 = 추상 자료형에서 정의된 연산
- 메소드의 호출 = 생성자

```java
interface Shape {
    void draw(); // 구체적인 방법은 하위 클래스에서 결정
}
```
