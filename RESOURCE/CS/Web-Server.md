# 웹 서버

웹 서버는 웹 컨텐츠 배포를 위해 만든 프로토콜이다.  
HTTP 또는 HTTPS를 통해 클라이언트 요청을 수행하는 서비스 프로그램 및 기반 하드웨어다.

웹 서버의 주된 기능은 웹 페이지를 클라이언트로 전달하는 것이다.

## 동작 방식

웹 서버는 웹 사이트의 코드와 데이터를 호스팅하는 기술이다.  
브라우저에 URL을 입력할 때 이 URL은 실제로 웹 서버의 주소 식별자이다.

브라우저와 웹 서버는 다음과 같이 통신한다.

1. 브라우저는 URL을 사용하여 서버의 IP 주소를 찾습니다.
2. 브라우저는 정보에 대한 HTTP 요청을 보냅니다.
3. 웹 서버는 데이터베이스 서버와 통신하여 관련 데이터를 찾습니다.
4. 웹 서버는 HTTP 응답으로 HTML 페이지, 이미지, 비디오 또는 파일과 같은 정적 콘텐츠를 브라우저에 반환합니다.
5. 그러면 브라우저가 정보를 표시합니다.
