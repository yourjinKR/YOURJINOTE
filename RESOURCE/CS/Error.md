
# Error

### 사전적 의미
- 부정확하거나 유효하지 않은 동작
- 그릇되어 이치에 맞지 않은 일

### 프로그래밍
- 프로그램이 의도대로 작동하지 않거나, 아예 실행되지 못하게 만드는 모든 문제나 실수
- 주요 프로그래밍 에러는 아래와 같이 구분한다
	- 컴파일 에러 (Compile Error)
	- 런타임 에러 (예외)
	- 논리적 에러

## Compile Error

소스코드를 컴파일할 때 발생하는 에러

## Runtime Error

프로그램 실행 도중 일어나는 에러

## Logical Error

실행은 되지만. 의도와 다르게 동작하는 것
