
# 제네릭을 사용하는 이유는?

[제네릭](Java-Generics.md)을 사용하는 목적은 **컴파일 시점에 타입 안정성을 확보**하고 **형변환을 제거**하여 **안전하고 재사용 가능한 코드**를 작성하기 위해서 입니다.  
제네릭을 사용하면 컬렉션이나 메서드가 어떤 타입을 다루는지 명확해지기 때문에 잘못된 타입이 들어오는 오류를 컴파일 타임에 잡을 수 있습니다.
그 결과 런타임에 발생하는 `ClassCastException`을 예방 할 수 있고 불필요한 캐스팅 코드를 제거할 수 있습니다.  
또한 **타입만 다르고 로직은 동일한 코드**를 하나의 구현으로 일반화할 수 있어 코드 중복을 줄이고 유지보수성을 높일 수 있습니다.

## 짧게

타입만 다르고 로직은 동일한 코드를 하나의 구현으로 일반화 시킬 수 있습니다.
